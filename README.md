NutriPro CRM Analytics: Biotech-Driven Retention Strategy
🧬 Project Overview
NutriPro là giải pháp phân tích dữ liệu CRM chuyên sâu dành cho nền tảng sức khỏe và dinh dưỡng. Dự án này kết hợp giữa mô hình phân tích dữ liệu hiện đại và kiến thức chuyên môn về Công nghệ sinh học (Biotechnology) để giải quyết bài toán cốt lõi: Giảm tỷ lệ khách hàng rời bỏ (Churn Risk) dựa trên cơ chế thích nghi sinh học..

Dự án tập trung vào việc phân tích hành vi người dùng đối với các chế độ ăn chuyên biệt (Keto, Vegan, High-protein) và xây dựng chiến lược can thiệp CRM cá nhân hóa.

📊 Key Data Insights (The Biotech Advantage)
Thay vì chỉ báo cáo con số, hệ thống phân tích này sử dụng góc nhìn của một Biotech Engineer để giải thích dữ liệu:

Keto Adaptation Window: Dữ liệu cho thấy rủi ro rời bỏ cao nhất xảy ra ở tuần thứ 2-3 do quá trình thích nghi chuyển hóa. Một quy trình CRM tự động đã được đề xuất để hỗ trợ người dùng vào đúng "điểm rơi" này.

Dietary Protocol Prediction: Các chỉ số Churn Risk Score được tính toán riêng biệt cho từng loại hình dinh dưỡng, giúp đội ngũ Coach can thiệp chính xác hơn.

🛠 Tech Stack
Data Processing: SQL, R, Python (Pandas).

Data Visualization: Power BI (Modern minimalist layout).

Analysis Model: RFM Analysis, Churn Prediction Scoring.

Version Control: GitHub.
